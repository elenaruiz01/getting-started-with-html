Basic Vocabulary

- HTML: HTML (Hypertext Markup Language) documents contain text content which describes "what you see" and markup which describes "how it looks".

- Tag: An HTML tag is always contained within angled brackets. Most tags have an opening tag (<p> for example) and a closing tag (</p>). Some tags (called "void" tags) do not require a closing tag (like the <br> tag).

- Attribute: This is a property of an HTML element. For example, to set the <href> attribute of an anchor tag to the Udacity URL, you would write <a href="www.udacity.com">

- Element: An HTML element refers to everything within a set of opening and closing tags. HTML documents are made of HTML elements. When writing HTML, we tell browsers the type of each element by using HTML tags.
(a) Container elements are elements that do nothing but contain stuff inside themselves.
(b) Inline elements: a, span, br, img, strong.
(c) Block elements: div, p, form.
